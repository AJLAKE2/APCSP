This is a p tag. Its purpose is to indicate to a browser that this text is considered a paragraph. Different browsers will render this content in sightly differnt ways, so it is likely to look different between browsers. We can also verride the applied default styling by using css.

This is a div. Divs are generic tags that can apply to anything. Browsers generally do not apply any default styling to divs. Before HTMLS became the web standard, divs were used for virtually everything. Because they are so generic it is hard to target them with CSS to apply styling, so when HTMLS became the standard, many new tags were added that make styling and structure easier.
